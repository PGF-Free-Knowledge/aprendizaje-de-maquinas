Jupyter notebook

Jupyter notebook es un en entorno de desarrollo interactivo (IDE) web, permite crear documentos con código en un formato bastante amigable, se puede programar por módulos o cajitas, imprimir el output del código en el mismo documento, comentar de forma amigable utilizando Markdown, LaTex, HTML, etc.

Hoja de trucos Markdown.

Algunos shortcuts:

Modo comando: Esc
Modo edición: Enter
Seleccionar multiples celdas: Shift-Down/UP
Ejecutar Celda/s: Ctrl+Enter
Guardar: Ctrl+S
Insertar celda de código arriba:A
Insertar celda de código abajo: B
Borrar Celda: D+D
Deshacer la última acción de las celdas: Z
Convertir a celda código: Y
Convertir a celda markdwon: M
Más shorcuts en Help > Keyboard Shorcuts.
